**Задание:** написать парсер для amazin.su

Парсинг каталога товаров в категории зоотовары. По каждому параметру при его наличии:
- название, 
- идентификатор, 
- артикул, 
- весовая и ценовая сетка (с учётом скидки), 
- производитель, 
- картинки, 
- гарантия, 
- единицы, 
- вес, 
- описание, 
- инструкции, 
- характеристики, 
- наличие, 
- вложенность 
- категорий, 
- отзывы.

Периодичность сбора: 1 раз в 5 дней.
